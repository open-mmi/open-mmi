## Notes

- The PNG files are dashboard-sized raster exports of the corresponding
  Wikimedia Commons SVG files listed above.
- Some upstream file pages note that the leading identifier, such as `A01`,
  `A16L`, or `B02`, is an ISO 2575 symbol code. This notice is only for the
  Wikimedia Commons artwork/licence; it is not a licence grant from ISO and does
  not redistribute the ISO standard itself.
- Keep this notice with the icon files when redistributing them. If any icon is
  modified, note that it was modified and comply with the CC BY-SA 3.0
  attribution and share-alike terms for the adapted asset.
